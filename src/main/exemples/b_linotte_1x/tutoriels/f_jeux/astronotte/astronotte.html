Entre Space Invaders et Arkanoid, Astronotte est un petit jeu qui vous amene loin loin loin...
Apres vous etes perdu, vous vous retrouvez dans une zone interdite appartenant a l'ennemi !
Votre mission : ressortir de ce terrain fort hostile afin de repartir tranquillement chez vous !
Astuce : vous pouvez utiliser la touche [T] pour tirer (plus discret que le 'clic' de la souris au boulot ...)
Le code source du jeu est librement disponible sur le forum
